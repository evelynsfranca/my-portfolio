Desenvolvido como projeto final de curso, o site D’gusto La Pizza foi criado para digitalizar e otimizar o gerenciamento de uma pizzaria, permitindo tanto a interação dos clientes com os serviços quanto o gerenciamento completo por parte do administrador.

Objetivo

O site busca simplificar a experiência dos clientes ao permitir que visualizem o cardápio online e realizem pedidos diretamente pelo site, enquanto os administradores têm acesso a ferramentas completas para a gestão de produtos, pedidos e promoções.

Esse projeto foi uma oportunidade prática para aplicar e consolidar os conhecimentos adquiridos durante a graduação, abrangendo desde a análise de requisitos até a entrega de uma solução funcional e bem estruturada.

Funcionalidades

Para clientes

  • Cadastro e login
  • Gerenciamento de dados pessoais e endereços
  • Adicionar produtos ao carrinho e realizar pedidos
  • Histórico e acompanhamento de pedidos

Para administradores

  • Gerenciamento de clientes e funcionários
  • Cadastro e edição de produtos e promoções
  • Controle e acompanhamento de pedidos

Desenvolvimento

O projeto foi realizado em equipe com três integrantes, cada um responsável por áreas específicas. Minha atuação incluiu:

  1. Desenvolvimento do Back-End

    Utilização de Spring Boot (Java) para construir APIs robustas e escaláveis

  2. Modelagem e Desenvolvimento do Banco de Dados

    Estruturado em MySQL para garantir eficiência e confiabilidade no armazenamento dos dados

  3. Infraestrutura

    • Deploy da aplicação utilizando Vercel (frontend)
    • Hospedagem da API no serviço EC2 da AWS, garantindo alta disponibilidade
